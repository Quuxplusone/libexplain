Find the Final Entry

The lookup of the final component of the pathname goes just like that of all other components, with two differences:
  1. The final component must not be a directory, unless it must be.
    opendir(pathname = "/home/pmiller/.netrc") failed, Not a directory (20, ENOTDIR) because in the pathname "/home/pmiller" directory there is a ".netrc" regular file, but it should be a directory
It all depends on the system call.
